## Latihan 2
Program untuk menampilkan bilangan terbesar dari n buah data yang di inputkan
Code :

![Code](Foto/Code_Latihan2_Praktikum3.png)

    max = 0

    while True :
        a = int(input("Masukkan Bilangan : "))
        if max < a :
            max = a
        if a == 0 :
            break
    print("Bilangan Terbesar Adalah : ", max)
